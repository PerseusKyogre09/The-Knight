## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/PerseusKyogre09/The-Knight.git
    ```
2. Navigate to the project directory:
    ```sh
    cd The-Knight
    ```
3. Compile the project:
    ```sh
    javac -d bin src/*.java
    ```
4. Run the application:
    ```sh
    java -cp bin Main
    ```s
